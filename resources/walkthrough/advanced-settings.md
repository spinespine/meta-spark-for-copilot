Customize base URL, model IDs, reasoning effort, and more in the extension settings.

- `Meta Spark: Open Settings`: Open settings
- `meta-spark-copilot.baseUrl`: default https://api.meta.ai/v1
- `meta-spark-copilot.maxCompletionTokens`: max output tokens
- Reasoning effort: minimal, low, medium (default), high, xhigh via model config dropdown
- Native vision: drop images, no proxy needed
