Muse Spark 1.1 for Copilot Chat uses your own Meta API key to make Muse Spark 1.1 available in the model picker.

Paste it once, then update or remove it later from the Command Palette.

- `Cmd/Ctrl + Shift + P`: Open the Command Palette
- `Meta Spark: Set API Key`: Set or update your API key (format LLM...)
- `Meta Spark: Clear API Key`: Remove your API key
- `Meta Spark: Get API Key`: Create a Meta API key at https://dev.meta.ai/
